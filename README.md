## Overview

Roblox Executor is a native Windows utility that loads and runs custom Lua and Luau scripts inside Roblox games. It is aimed at users who want a straightforward workflow without heavy installers, multi-step setup, or unnecessary extras. The interface is kept deliberately simple so you can get from launch to script execution quickly.

The program is designed to stay lean on system resources while remaining compatible with current Roblox client updates. A bundled script hub gives you instant access to hundreds of community-made scripts, spanning simple convenience tweaks through more advanced game modifications. There are no accounts to create and no paid tiers to unlock - just download it, open it, and begin experimenting.

## FAQ

**Is this tool safe to use?**  
Use it at your own risk. Third-party tools that alter Roblox gameplay can conflict with Roblox's terms of service. This project is offered for educational and testing use only.

**Will it work after Roblox updates?**  
Yes, updates are monitored so compatibility can be refreshed when the Roblox client changes. For the newest release, check the repository.

**How does this compare to other executors?**  
The focus here is on simplicity and a small footprint. There are no unnecessary extras or paid unlocks - everything is free and available right away.

**Can my Roblox account get banned?**  
There is always some risk when using script execution tools. For testing, an alternate account is recommended, and public servers should be approached carefully.

**Where are my scripts stored?**  
Downloaded scripts and your own files are kept locally inside `ScriptHub/`. Nothing is uploaded or shared unless you choose to do so.
